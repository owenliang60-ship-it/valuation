# Sensitivity Tables

> 不同市场 Regime 下的 OPRMS 仓位敏感性分析
> 基准假设: 总资产 $5,000,000

---

## 1. DNA x Timing Matrix ($ Amount)

基于 $5M 总资产的单只股票仓位金额：

| | Timing S (1.25) | Timing A (0.9) | Timing B (0.5) | Timing C (0.2) |
|---|---|---|---|---|
| **DNA S (25%)** | $1,562,500 | $1,125,000 | $625,000 | $250,000 |
| **DNA A (15%)** | $937,500 | $675,000 | $375,000 | $150,000 |
| **DNA B (7%)** | $437,500 | $315,000 | $175,000 | $70,000 |
| **DNA C (2%)** | $125,000 | $90,000 | $50,000 | $20,000 |

*Timing S 使用中值 1.25 (范围 1.0-1.5)*
*Timing A 使用中值 0.9 (范围 0.8-1.0)*
*Timing B 使用中值 0.5 (范围 0.4-0.6)*
*Timing C 使用中值 0.2 (范围 0.1-0.3)*

## 2. DNA x Timing Matrix (% of Portfolio)

| | Timing S (1.25) | Timing A (0.9) | Timing B (0.5) | Timing C (0.2) |
|---|---|---|---|---|
| **DNA S** | 31.3% → cap 25% | 22.5% | 12.5% | 5.0% |
| **DNA A** | 18.8% → cap 15% | 13.5% | 7.5% | 3.0% |
| **DNA B** | 8.8% → cap 7% | 6.3% | 3.5% | 1.4% |
| **DNA C** | 2.5% → cap 2% | 1.8% | 1.0% | 0.4% |

> 当 Timing S 系数 > 1.0 时，计算结果可能超过 DNA 硬上限。此时以 DNA 上限为准（S=25%, A=15%, B=7%, C=2%）。

## 3. Regime Scenarios

### Bull Market Regime

**特征**: VIX 12-18, 利率下行或持稳, 经济扩张, 企业盈利增长

| 调整 | 变化 |
|------|------|
| 总仓位 | 可达 90-95% |
| Timing 偏移 | 多数持仓 Timing A-S |
| 典型组合 | 2x S-tier + 3x A-tier + 2x B-tier |
| 现金 | 5-10% |

**示例组合** ($5M):

| Position | DNA | Timing | % | Amount |
|----------|-----|--------|---|--------|
| Stock 1 | S | A (0.9) | 22.5% | $1,125,000 |
| Stock 2 | S | B (0.5) | 12.5% | $625,000 |
| Stock 3 | A | A (0.9) | 13.5% | $675,000 |
| Stock 4 | A | A (0.8) | 12.0% | $600,000 |
| Stock 5 | A | B (0.5) | 7.5% | $375,000 |
| Stock 6 | B | A (0.9) | 6.3% | $315,000 |
| Stock 7 | B | B (0.5) | 3.5% | $175,000 |
| **Cash** | | | **22.2%** | **$1,110,000** |
| **Total** | | | **100%** | **$5,000,000** |

### Base Case Regime

**特征**: VIX 18-25, 利率中性, 经济温和增长, 盈利平稳

| 调整 | 变化 |
|------|------|
| 总仓位 | 70-85% |
| Timing 偏移 | 多数持仓 Timing B |
| 典型组合 | 1-2x S-tier + 2x A-tier + 1-2x B-tier |
| 现金 | 15-30% |

### Bear Market Regime

**特征**: VIX 25-40, 利率上行, 经济收缩, 盈利下修

| 调整 | 变化 |
|------|------|
| 总仓位 | 40-60% |
| Timing 偏移 | 多数持仓 Timing C, 仅核心持 B |
| 典型组合 | 1x S-tier + 1x A-tier, 可能有做空 |
| 现金 | 40-60% |

**示例组合** ($5M):

| Position | DNA | Timing | % | Amount |
|----------|-----|--------|---|--------|
| Stock 1 | S | C (0.3) | 7.5% | $375,000 |
| Stock 2 | A | C (0.2) | 3.0% | $150,000 |
| Short 1 | B | B (0.5) | -3.5% | -$175,000 |
| **Cash** | | | **93.0%** | **$4,650,000** |
| **Total** | | | **100%** | **$5,000,000** |

---

### Crisis Regime

**特征**: VIX > 40, 流动性枯竭, 系统性风险, 恐慌性抛售

| 调整 | 变化 |
|------|------|
| 总仓位 | 20-40% (仅 S-tier 核心) |
| Timing 偏移 | 既有持仓 Timing C；但市场恐慌底可能触发 Timing S |
| 策略 | 极度防守 → 如确认底部则转为大胆加仓 |
| 现金 | 60-80% |

**恐慌底部加仓场景** (VIX > 40, 确认结构性底部):

| Position | DNA | Timing | % | Amount |
|----------|-----|--------|---|--------|
| Stock 1 | S | S (1.5) | 25.0% (cap) | $1,250,000 |
| Stock 2 | S | S (1.2) | 25.0% (cap) | $1,250,000 |
| Stock 3 | A | S (1.0) | 15.0% (cap) | $750,000 |
| **Cash** | | | **35.0%** | **$1,750,000** |
| **Total** | | | **100%** | **$5,000,000** |

> 此为极端场景。需满足：完整备忘录 + 全部 kill conditions + S-Timing 证据充分。"千载难逢"一年不会出现一次。

## 4. Drawdown Impact on Sizing

当组合层面触发回撤协议时，仓位上限进一步收紧：

| 回撤阶段 | DNA S 上限 | DNA A 上限 | DNA B 上限 | DNA C 上限 | 总仓位上限 |
|---------|-----------|-----------|-----------|-----------|-----------|
| 正常 | 25% | 15% | 7% | 2% | 95% |
| -10% 排查 | 25% | 15% | 5% | 0% | 85% |
| -15% 防御 | 20% | 12% | 0% | 0% | 70% |
| -20% 危机 | 15% | 0% | 0% | 0% | 50% |

---

## 5. Maximum Simultaneous Positions

基于集中度限制和最小有效仓位 (0.5%)：

| DNA Rating | 理论最大同时持有数 | 建议最大 |
|-----------|------------------|---------|
| S | 2 (硬限制) | 2 |
| A | 6 (=100%/15%, 但受总仓位限制) | 4 |
| B | 14 | 5 |
| C | 50 | 3 |

**建议总持仓数**: 7-15 只，确保每只都有足够关注度和研究深度。
